## 使用说明

新的设计系统已经集成到现有组件中，所有样式都通过CSS类应用。主要的设计令牌可以在 `global.css` 中找到，包括：

- 颜色变量：`--primary-*`, `--text-*`, `--bg-*`
- 间距变量：`--space-*`
- 圆角变量：`--radius-*`
- 阴影变量：`--shadow-*`
- 渐变变量：`--gradient-*`

所有组件都支持暗色主题，通过 `data-theme="dark"` 属性自动切换。
