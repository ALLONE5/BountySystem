## 🚀 如何使用

### 1. 查看效果
打开 `test-discord-ui.html` 查看Discord风格效果预览

### 2. 启动应用
```bash
# 启动后端
cd packages/backend
npm run dev

# 启动前端
cd packages/frontend
npm run dev
```

### 3. 设置主题
在浏览器中访问应用后，主题会自动设置为Discord风格

### 4. 切换主题
可以在设置页面中切换不同主题：
- Discord (默认)
- Midjourney
- Dark
- Light
- Cyberpunk
