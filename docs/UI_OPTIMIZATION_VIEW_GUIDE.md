## 设计系统体验

### 色彩系统
观察以下颜色的使用：
- **主色（蓝色 #1890ff）**: 主要按钮、链接
- **成功（绿色 #52c41a）**: 完成状态、正向反馈
- **警告（橙色 #faad14）**: 进行中、需注意
- **错误（红色 #f5222d）**: 赏金金额、危险操作

### 间距系统
注意以下间距的一致性：
- 页面容器：24px padding
- 卡片间距：16px gap
- 元素间距：8px-16px

### 圆角系统
观察不同元素的圆角：
- 卡片：4px
- 按钮：2px
- 输入框：2px

### 阴影系统
注意不同层级的阴影：
- 卡片：轻微阴影
- 悬停：中等阴影
- 模态框：较深阴影
