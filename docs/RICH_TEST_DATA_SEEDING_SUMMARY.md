## 脚本文件

### 主脚本
- **文件**: `packages/backend/scripts/seed-rich-test-data.js`
- **功能**: 清理现有数据并注入新的测试数据
- **执行**: `node packages/backend/scripts/seed-rich-test-data.js`

### 验证脚本
- **文件**: `packages/backend/scripts/verify-rich-test-data.js`
- **功能**: 验证数据注入结果，显示统计信息
- **执行**: `node packages/backend/scripts/verify-rich-test-data.js`

## 技术实现

### ES Module 转换
- 将脚本从 CommonJS 转换为 ES Module
- 使用 `import` 替代 `require`
- 使用 `crypto.randomUUID()` 生成 UUID

### UUID 生成
- 使用 Node.js 内置的 `crypto.randomUUID()` 生成 UUID
- 解决了数据库 `uuid_generate_v4()` 函数可能不可用的问题

### 数据清理
- 在注入新数据前清理现有的任务和项目组
- 使用事务确保数据一致性
