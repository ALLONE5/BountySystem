## 常见问题

### Q1: 主题没有生效

**解决方案**:
1. 清除浏览器缓存 (Ctrl+Shift+Delete)
2. 强制刷新 (Ctrl+F5)
3. 检查浏览器控制台是否有错误
4. 重启前端服务

### Q2: 表格文字看不清

**解决方案**:
1. 确保选择了正确的主题
2. 检查浏览器缩放级别 (应为100%)
3. 尝试不同的浏览器
4. 检查显示器亮度设置

### Q3: 按钮没有发光效果

**解决方案**:
1. 确保选择了赛博朋克主题
2. 检查浏览器是否支持CSS阴影
3. 尝试不同的浏览器
4. 检查浏览器硬件加速是否启用

### Q4: 主题切换时有闪烁

**解决方案**:
1. 这是正常的，因为需要重新加载页面
2. 如果闪烁过长，检查网络连接
3. 尝试清除浏览器缓存
