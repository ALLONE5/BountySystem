## 🎨 设计特点

### 统一的设计语言
所有页面遵循相同的设计模式：
1. **标题栏**：页面标题 + 操作按钮（如需要）
2. **搜索筛选栏**：Search + Select 组件
3. **数据表格**：简洁的列，只显示关键信息
4. **详情展示**：Drawer 或 Modal 显示完整信息
5. **操作按钮**：统一的图标和样式

### 颜色系统
- **主色**: #1890ff (蓝色)
- **成功**: #52c41a (绿色)
- **警告**: #faad14 (橙色)
- **错误**: #f5222d (红色)
- **默认**: #d9d9d9 (灰色)

### 组件使用
- **Typography.Title**: 页面标题
- **Card**: 内容容器
- **Table**: 数据列表
- **Tag**: 状态标签
- **Button**: 操作按钮
- **Modal**: 编辑/确认对话框
- **Drawer**: 详情展示
- **Form**: 表单输入
- **Descriptions**: 详情描述
- **Progress**: 进度条
- **Tabs**: 标签页

## 📝 使用说明

### 用户管理页面
1. 在搜索框输入用户名或邮箱进行搜索
2. 使用角色下拉框筛选不同角色的用户
3. 点击用户名查看详细信息
4. 点击"编辑"按钮修改用户信息
5. 点击"删除"按钮删除用户（仅超级管理员）

### 任务管理页面
1. 在搜索框输入任务名称或描述进行搜索
2. 使用状态下拉框筛选不同状态的任务
3. 点击任务名查看详细信息
4. 点击"编辑"按钮修改任务信息
5. 点击"删除"按钮删除任务

### 审核操作页面
1. 使用 Tab 标签切换不同状态的申请
2. 点击"查看"按钮查看申请详情
3. 在表格中点击"批准"快速批准申请
4. 点击"拒绝"按钮打开审核模态框，填写拒绝理由
5. 已审核的申请可以查看但无法再次操作
