## 🎨 设计特点

### 统一的设计语言
所有页面采用相同的设计模式：
1. **标题栏**: 页面标题
2. **搜索筛选**: Search + Select 组件
3. **数据表格**: 简洁清晰的列
4. **详情展示**: Drawer 或 Modal
5. **操作按钮**: 统一的图标和样式

### 彩色标签系统
- **用户角色**: 灰色普通用户 / 蓝色职位管理员 / 红色超级管理员
- **任务状态**: 灰色未开始 / 绿色可承接 / 蓝色进行中 / 紫色已完成 / 红色已放弃
- **可见性**: 蓝色公开 / 橙色仅岗位 / 红色私有
- **申请状态**: 橙色待审核 / 绿色已批准 / 红色已拒绝

### 组件使用
- **Typography.Title**: 页面标题
- **Card**: 内容容器
- **Table**: 数据列表
- **Tag**: 状态标签
- **Button**: 操作按钮
- **Modal**: 编辑/确认对话框
- **Drawer**: 详情展示
- **Form**: 表单输入
- **Descriptions**: 详情描述
- **Progress**: 进度条
- **Tabs**: 标签页

## 📝 使用说明

### 用户管理
1. 在搜索框输入用户名或邮箱
2. 使用角色下拉框筛选
3. 点击用户名查看详情
4. 点击"编辑"修改信息
5. 点击"删除"删除用户（仅超级管理员）

### 任务管理
1. 在搜索框输入任务名称或描述
2. 使用状态下拉框筛选
3. 点击任务名查看详情
4. 点击"编辑"修改信息
5. 点击"删除"删除任务

### 审核操作
1. 使用 Tab 切换不同状态
2. 点击"查看"查看详情
3. 点击"批准"快速批准
4. 点击"拒绝"填写理由并拒绝
5. 已审核的申请可查看但无法再次操作
