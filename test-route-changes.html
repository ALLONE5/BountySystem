🔧 故障排除

如果页面没有变化，请尝试以下方法：

  1. 强制刷新浏览器：按 Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)
  2. 清除浏览器缓存：
    • 按 F12 打开开发者工具
    • 右键点击刷新按钮
    • 选择"清空缓存并硬性重新加载"
  3. 重启开发服务器：
    • 在终端中按 Ctrl+C 停止服务器
    • 运行：cd packages/frontend && npm run dev
  4. 检查控制台错误：按 F12 查看是否有 JavaScript 错误
