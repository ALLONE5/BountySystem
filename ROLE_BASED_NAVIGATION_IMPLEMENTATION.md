# 🎯 基于角色的分级导航系统实现

## 📱 新的导航结构

### 普通用户 (3个模块)
1. **我的** 👤 - 个人界面、我的悬赏、我的任务、我的组群
2. **赏金任务** 💰 - 浏览任务、任务邀请
3. **猎人排名** 🏆 - 排名系统

### 管理员 (4个模块)
1. **我的** 👤 - 个人相关功能
2. **赏金任务** 💰 - 任务相关功能  
3. **猎人排名** 🏆 - 排名功能
4. **管理** ⚙️ - 用户管理、组群管理、任务管理等

### 开发者 (5个模块)
1. **我的** 👤 - 个人相关功能
2. **赏金任务** 💰 - 任务相关功能
3. **猎人排名** 🏆 - 排名功能
4. **管理** ⚙️ - 管理功能
5. **开发** 🔧 - 系统配置、审计日志

## 🔧 技术实现

### 1. 权限检查系统
- 使用 `usePermission` hook 检查用户角色
- 动态显示/隐藏导航模块
- 支持多角色权限判断

### 2. 模块化页面设计
- **MyPage.tsx**: "我的"模块整合页面
- **BountyTasksPage.tsx**: "赏金任务"模块整合页面
- 使用 Tabs 组件实现子功能切换

### 3. 路由优化
- 新增模块化路由: `/my`, `/bounty-tasks`
- 保持原有路由兼容性
- 智能路径匹配和导航高亮

### 4. 底部导航栏
- 动态按钮数量 (3-5个)
- 基于权限的条件渲染
- 现代化图标和交互设计
